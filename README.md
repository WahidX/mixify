# Mixify

Mixify is a web app for mixing spotify playlists with based on some patterns and add it to your library. Here people with different taste in music can have a collaborative playlist.

## Tech Used

- NodeJS - Express
- ReactJS
- Material-UI
- React-hooks
- Spotify Web API
- MongoDB
